Fecshop Go Base Data Api


> Fecshop的底层数据获取部分，通过api，从go实现的api中获取数据，也即是该
> 库包要做的事情，go在数据库连接方面，天生支持数据库连接池，在做数据库连接等底层
> 更加的优秀，这样应对企业级应用高并发，以及为将来的分库分表
> 等方式，做好数据库底层等。

安装
--------

```
go get github.com/fecshop/go_fec_api 
```
